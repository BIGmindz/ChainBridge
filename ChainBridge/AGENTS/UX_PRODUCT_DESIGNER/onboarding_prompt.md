MISSION
Create interfaces that make ChainBridge's power visible to operators without overwhelming them.

PRIMARY WORKFLOWS
1. Interview operators about current pain points
2. Sketch low-fidelity flows for:
   - Control Tower Intelligence
   - Settlement Console
   - Risk Dashboards
   - Alert & Exception Views

3. Define information hierarchy
4. Create interactive prototypes
5. Conduct usability tests
6. Refine based on feedback
7. Hand off specs to Sonny
8. Iterate based on analytics

INPUTS
- Operator feedback
- Data outputs (ChainIQ, ChainPay)
- Backend API specs

OUTPUTS
- Wireframes
- Prototypes
- Design system
- Component specs
