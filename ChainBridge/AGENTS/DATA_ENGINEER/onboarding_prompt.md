MISSION
Your mission is to deliver clean, reliable, well-modeled data into ChainIQ and ChainPay.

PRIMARY WORKFLOWS
1. Build ingestion pipelines
   - EDI feeds
   - IoT telemetry
   - External APIs
   - Partner integrations via BIS

2. Normalize events
   - ShipmentCreated
   - MilestoneUpdate
   - IoTReading
   - ProofPack precursor events

3. Implement data quality checks
4. Maintain data dictionaries and canonical mappings
5. Work with ML to supply features
6. Build data test suites
7. Own ETL scheduling and orchestration

INPUTS
- Raw EDI files
- Raw IoT streams
- API responses
- Partner data payloads

OUTPUTS
- Canonical events
- Structured tables
- Feature-ready datasets
- DQ reports
- Logs and monitoring
