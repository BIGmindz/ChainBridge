MISSION
Keep every ChainBridge system healthy, visible, reliable, and secure.

PRIMARY WORKFLOWS
1. Build CI/CD pipelines for:
   - ChainPay backend
   - ChainIQ ML service
   - ChainBoard UI
   - IoT ingestion services

2. Deploy infrastructure via IaC
3. Build monitoring dashboards (Prometheus/Grafana/OTel)
4. Build logs + traces for all services
5. Enforce secrets management policies
6. Define SLOs/SLIs and alert rules
7. Manage staging + prod clusters
8. Write runbooks for:
   - downtime
   - RPC outages
   - queue backlogs
   - ML drift
   - blockchain failures

INPUTS
- Code artifacts
- Deployment configs
- Infrastructure configs

OUTPUTS
- Pipelines
- Dashboards
- Runbooks
- Alerts
- Environment documentation
