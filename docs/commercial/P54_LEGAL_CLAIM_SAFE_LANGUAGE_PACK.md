## 2. Core Language Rules

### 2.1 Forbidden Terms (NEVER USE)

| Category | Forbidden | Reason |
|----------|-----------|--------|
| Certification | "certified", "certification", "certify" | Implies regulatory status |
| Guarantees | "guaranteed", "guarantee", "assured" | Creates liability |
| Security Absolutes | "secure", "impenetrable", "breach-proof" | Impossible to guarantee |
| Compliance | "compliant", "compliance certified" | Implies regulatory attestation |
| Risk Elimination | "risk-free", "zero risk", "eliminates risk" | Misleading |
| Perfection | "100% accurate", "never fails", "always correct" | Impossible |

### 2.2 Required Replacements

| Forbidden | Required Alternative |
|-----------|---------------------|
| "certified API" | "verified API" |
| "security guaranteed" | "security-tested" |
| "compliant with SOC2" | "tested against SOC2 controls" |
| "eliminates risk" | "helps identify risks" |
| "100% secure" | "thoroughly tested" |
| "breach-proof" | "defense-in-depth approach" |

## 3. Approved Language Templates

### 3.1 Product Description

**APPROVED:**
```
ChainBridge provides API verification services that assess and score 
the behavior of third-party APIs against defined test criteria. 
ChainVerify generates verification reports based on automated testing.
These reports reflect observed behavior during testing and do not 
constitute certification, compliance attestation, or security guarantees.
```

**BLOCKED:**
```
ChainBridge certifies APIs and guarantees their security. Our 
certification ensures compliance with industry standards.
```

### 3.2 Trust Score Description

**APPROVED:**
```
The ChainVerify Trust Score is a numerical assessment based on 
automated tests against defined criteria. Scores reflect observed 
behavior at the time of testing. Scores are not guarantees of 
future behavior, security status, or regulatory compliance.
```

**BLOCKED:**
```
The ChainVerify Trust Score certifies that an API is secure and 
compliant. Higher scores guarantee better security.
```

### 3.3 Sales Language

**APPROVED:**
```
"ChainVerify helps you assess API behavior through automated testing"
"Our verification reports provide visibility into API characteristics"
"ChainBridge enables continuous monitoring of API behavior"
"Trust scores reflect test results, not security guarantees"
```

**BLOCKED:**
```
"ChainVerify certifies your APIs"
"Our reports guarantee security"
"ChainBridge ensures your APIs are compliant"
"High trust scores mean your API is secure"
```

ChainBridge provides verification services, not certification services.
All scores, grades, and reports reflect automated test results only.

## 6. Contract Language

### 6.1 Service Description (Approved)

```
ChainBridge shall provide Verification Services consisting of:
(a) Automated testing of designated APIs against defined test criteria
(b) Generation of verification reports reflecting test results
(c) Assignment of numerical scores based on test outcomes

Verification Services do not include and shall not be construed as:
(a) Certification of any kind
(b) Compliance attestation or audit
(c) Security guarantee or warranty
(d) Penetration testing or security assessment
```

### 6.2 Limitation of Liability (Approved)

```
IN NO EVENT SHALL CHAINBRIDGE BE LIABLE FOR ANY INDIRECT, INCIDENTAL,
SPECIAL, CONSEQUENTIAL, OR PUNITIVE DAMAGES ARISING FROM OR RELATED TO
THE VERIFICATION SERVICES OR ANY VERIFICATION REPORTS, INCLUDING BUT NOT
LIMITED TO DAMAGES ARISING FROM SECURITY INCIDENTS, DATA BREACHES, OR
REGULATORY ACTIONS AFFECTING APIS THAT HAVE BEEN VERIFIED.
```
